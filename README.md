# Volleyball DJ Pult (statische PWA)

- DJ-Pult für die Volleyball-Halle: Musik-Snippets nach Kategorien abspielen (Play, Stop mit Fade, Lautstärke).
- Wichtige Dateien: `index.html` (UI), `script.js` (Logik & Button-Aufbau), `style.css` (Styles), `service-worker.js` (Offline-Cache), `manifest.json` (App-Metadaten), `static/` (music/, special_music/, images/).
- Lokal testen: Im Ordner `pwa/` starten – `python -m http.server 8000` – dann im Browser `http://localhost:8000` öffnen.
- Hosting: Ordner `pwa/` kann 1:1 auf statischen Hostern (z. B. GitHub Pages) genutzt werden.
